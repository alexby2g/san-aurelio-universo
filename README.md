# San Aurelio — Universo Completo

Este paquete une el Capítulo 0 y el Capítulo 1 en una sola estructura web lista para GitHub y Vercel.

## Estructura

- `index.html`: portada principal.
- `capitulos/capitulo_0_antes_del_silencio.html`
- `capitulos/capitulo_1_la_manana_despues.html`
- `assets/capitulo_0/`: imágenes del Capítulo 0.
- `assets/capitulo_1/`: imágenes del Capítulo 1.
- `textos/`: textos completos.

## Uso

Abre `index.html` en el navegador.

Para actualizar GitHub/Vercel, reemplaza los archivos de tu proyecto por estos y ejecuta:

```bash
git add .
git commit -m "Une capitulos 0 y 1 de San Aurelio"
git push origin main
```
